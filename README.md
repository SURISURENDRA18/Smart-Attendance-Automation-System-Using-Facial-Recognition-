Smart Attendance Automation System Using Facial Recognition

The Smart Attendance Automation System is a computer vision–based application that automates attendance recording using facial recognition technology. The system captures video through a webcam, detects faces in real time, compares them with stored facial encodings, and automatically records attendance with timestamps.

This solution eliminates manual attendance processes and provides a fast, contactless, and reliable attendance management system suitable for classrooms, workplaces, and organizations.

Features
	•	Real-time face detection using webcam
	•	Face recognition using pre-trained facial encodings
	•	Automatic attendance marking with date and time
	•	CSV-based attendance storage
	•	Multiple user support
	•	Lightweight and easy to deploy

Technologies Used
	•	Python
	•	OpenCV
	•	face_recognition
	•	NumPy
	•	CSV for attendance storage
